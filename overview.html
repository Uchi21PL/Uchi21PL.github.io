Localization by Uchi21PL Help

Overview

This asset allows you to use Google Sheets as solution of storing and editing your game Localization. It's free to use, and will require small amount of effort to set up.

Requirements

Your sheet need's to be set as 'Anyone with the link' to allows this asset download automatically your localization. Also, this asset works only with TextMeshPro, there is no support to legacy Text.

Official support for Unity 6000+, but there is great probability that this will work with older versions.

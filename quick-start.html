Localization by Uchi21PL Help

Quick Start

To start using Localization, you need to create new blank Google spreadsheet

Set 'Anyone with the link'

You need to set access to the sheet. To do so, press Share button and set General access to Anyone with the link.

This will allow Localization to download directly from Google Sheets.

How Spreadsheet is used

Your spreadsheet will be used in following way:
First column will contain Keys, that will be used during Localization process.
Rest of columns will be separate languages, linked to this key.
So column A is key, and B will be first language, C will be the second Language, and so on

Setup in Unity

Find in Assets prefab of LocalizationManager.
It should be inside Assets\Localization\Resources.

You can also use Menu option: Uchi->Select Localization Prefab

You need to fill the Localization File URL field with your sharing link (press copy link button in the window where you was setting General access)

This will allow you to download Localization from main spreadsheet.

Now, you need to set up languages in Manager.

Add to Supported Languages list all languages that you will support.
Each one of them will have own column in spreadsheet, with first one starting as column B.

Downloading Localization

Now, just by pressing Download and Reload Localization button, you will download your Localization.
There is no need to do more, it's that simple.

Localization of TextMeshPro

Now, you need to add Localization component to GameObject with TextMeshPro, and fill theLocalized Keyfield with value from column A.

You can test if it works properly by pressingLoad localization cachceButton, that will get cache and display you localization.
By pressing Set to X you will change current text in your TextMeshPro to your localized value.
